Example 1:
>>> my_list = MyList()
>>> my_list.print_sorted()
[]

Example 2:
>>> my_list = MyList([1, 4, 2, 3, 5])
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]

Example 3:
>>> my_list = MyList([1, 4, 2, 3, 5])
>>> my_list.append(8)
>>> my_list
[1, 4, 2, 3, 5, 8]

Example 4:
>>> my_list = MyList([1, 4, 2, 3, -5])
>>> my_list.print_sorted()
[-5, 1, 2, 3, 4]
